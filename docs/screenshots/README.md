# Screenshots

ASCII captures of every PHOSPHOR screen, kept in plain text so they
diff cleanly under git and render correctly in any monospace font.
These are simulations rather than literal terminal grabs — they
reflect the current layout but use ASCII fallback glyphs so they
display anywhere this `README.md` does.

| Screen | File |
| ------ | ---- |
| HOME    | [`home.txt`](home.txt)       |
| SYS     | [`sys.txt`](sys.txt)         |
| MAIL    | [`mail.txt`](mail.txt)       |
| TIME    | [`time.txt`](time.txt)       |
| WEATHER | [`weather.txt`](weather.txt) |
| LOG     | [`log.txt`](log.txt)         |

For real screen captures of the phosphor-green and amber-crt themes
in a terminal of your choice, run `ph0sphor-client --demo` (or
`ph0sphor-server --demo` plus a paired client) and screenshot with
your terminal's native facility. We deliberately do not include
binary PNGs in the repo to keep it cheap to clone.
