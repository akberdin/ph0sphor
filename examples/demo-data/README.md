# Demo Data

This directory will hold canned telemetry snapshots used by:

- `ph0sphor-server --demo` to drive realistic fake telemetry.
- `ph0sphor-client --demo` to run without a real server.
- Integration tests and screenshot generation.

Demo data lands with the Milestone 2 demo data generator
(`ph0sphorctl gen-demo`). Until then, this directory is intentionally empty.
